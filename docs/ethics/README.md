# Ethics and data handling

- Public datasets remain subject to their original licenses and access conditions.
- Team recordings are limited to interface and pipeline validation unless a separate approved
  protocol explicitly permits research use.
- Raw face video, identity records, and consent forms are never committed to Git.
- Manifests use anonymous signer codes.
- The interface states that this is not a medical device and cannot make medical decisions.
- Emergency and pain intents use conservative templates that direct the user to staff; the
  semantic layer never invents a diagnosis.
